# V16 Proposal — 互動體驗升級

> 基於 V15 音頻-頁面聯動系統的基礎，V16 聚焦讓用戶從「觀看者」變成「參與者」。

## 📊 當前頁面現狀分析

### 已完成（V1-V15）
- ✅ 完整的品牌敘事結構（8 個章節）
- ✅ 高品質視覺設計（黑金風格、粒子背景、3D 卡片效果）
- ✅ AI 數字員工展示（Lawrence/Luke/Lawra/Lancelot）
- ✅ 寵物系統（Labubu 多語言語音）
- ✅ 音頻解說 + 字幕 + 關鍵詞聯動
- ✅ 定價方案 + ROI 計算展示

### 缺失的互動維度
- ❌ 用戶無法「體驗」Lawrence 的能力，只能「看到」描述
- ❌ 沒有個性化路徑 —— 所有訪客看到完全相同的內容
- ❌ 沒有數據輸入 —— 用戶無法輸入自己的場景來評估價值

## 🎯 V16 三大核心功能

### 1. AI 對話模擬器 (Lawrence Live Demo)

在「場景演示」章節嵌入一個可互動的聊天窗口：

```
┌─────────────────────────────────────┐
│  🛡️ Lawrence · 即時體驗            │
│─────────────────────────────────────│
│  選擇場景：                         │
│  [合約審查] [法規查詢] [隱私評估]    │
│─────────────────────────────────────│
│  👤 我們準備在深圳設立子公司，       │
│     需要了解當地的數據合規要求       │
│                                     │
│  🛡️ 根據《個人信息保護法》及深圳    │
│     經濟特區數據條例，您需要關注     │
│     以下三個核心合規要點：           │
│     1. 數據跨境傳輸...              │
│     2. 個人信息保護影響評估...       │
│     3. 數據安全等級保護...           │
│                                     │
│  ┌─────────────────────────────┐    │
│  │ 輸入您的問題... [發送]      │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

**技術方案：**
- 前端：純靜態模擬（預設 5-8 組對話），無需後端
- 打字機效果逐字輸出，模擬真實 AI 回覆
- 可選：接入真實 API（需評估成本和安全性）

### 2. ROI 計算器 (互動式)

替換當前靜態的 ROI 數據展示，讓用戶輸入自己的參數：

```
┌─────────────────────────────────────┐
│  💰 專屬 ROI 計算                   │
│─────────────────────────────────────│
│  團隊規模：    [▼ 5-10人 ]          │
│  合約處理量：  [▼ 50份/月]          │
│  法規覆蓋區域：[▼ 大中華 ]          │
│─────────────────────────────────────│
│  📊 預估年度節省                     │
│                                     │
│  人力成本節省    ¥180,000           │
│  時間效率提升    ×3.2               │
│  合規風險降低    -67%               │
│  ──────────────────                 │
│  預估總節省      ¥223,000/年        │
│                                     │
│  [獲取詳細報告 →]                   │
└─────────────────────────────────────┘
```

**技術方案：**
- 純前端計算，滑桿 + 下拉選單
- 數據動態更新，count-up 動畫
- 結果可生成簡易 PDF 或截圖分享

### 3. 場景導覽模式 (Guided Tour)

首次訪問時提供互動式導覽，根據用戶角色定制瀏覽路徑：

```
「歡迎！您的角色是？」
  [企業法務]  [管理層]  [合規專員]  [技術團隊]

→ 企業法務：重點展示 合約審查 → 法規檢索 → 場景演示
→ 管理層：重點展示 ROI → 安全體系 → 定價
→ 合規專員：重點展示 隱私保護 → 風險評估 → 全能力概覽
→ 技術團隊：重點展示 安全架構 → API 整合 → 部署方案
```

**技術方案：**
- Cookie/localStorage 記住選擇
- 對應章節添加 spotlight 效果
- 非重點章節折疊（可展開），減少信息負荷

## 🎨 附加增強

### 微互動升級
- 能力卡片懸停時顯示「點擊查看詳情」展開面板
- 定價卡片對比模式（左右滑動對比不同方案）
- CTA 按鈕增加 confetti 效果（點擊預約時）

### 移動端體驗
- 底部快速跳轉工具列（替代長頁面滾動）
- 卡片左右滑動切換（touch-friendly）
- 字幕和播放器合併為一體（節省螢幕空間）

### 數據追蹤
- 各章節停留時間統計
- 音頻播放完成率
- 互動元素點擊熱力圖
- 為後續優化提供數據支撐
